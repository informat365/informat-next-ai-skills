This document is the website module documentation for the Informat low-code platform.

## Overview
The Informat website and resource hosting feature provides static resource hosting services, server-side rendering (SSR) capabilities, and more, enabling quick deployment of static or dynamic websites.

## Resources and URLs
Creating a website module is equivalent to creating a site with a root directory of ${host}/web0/website/${appId}/${websiteId}/.
Where ${host} is the platform's webUrl which can be obtained by calling the `_app_get_web_url` tool, ${appId} is the application ID, and ${websiteId} is the website module ID.
If I upload an index.html file to the website module, its corresponding URL is ${host}/web0/website/${appId}/${websiteId}/index.html, and the relative path within the site is ./index.html

The appId is automatically generated by the system and may differ across environments. To ensure the application can access specified resources when distributed to any environment, you need to dynamically obtain the appId and set the resource path accordingly in code that references website resources.
For resources within the site that depend on each other, it is recommended to use relative paths.

## Example 2: Calling an API via main.js

The following example demonstrates calling the Informat low-code platform API from the website module via main.js, and rendering the returned data to the page.

The application has a created API endpoint:
Path: news
Return type: json
The API returns a news list
Display news title list on the frontend page

main.js

```js
/**
 * Website module main.js
 * Demonstrates how to call the Informat API from a frontend page
 */

// Informat service address (example: using the current site)
const host = window.location.origin;

// Application ID (hardcoded in this example; recommended to obtain dynamically in actual projects)
const appId = 'demoAppId123';

// API path
const apiPath = 'news';

// Full API address
const apiUrl = `${host}/web0/api/${appId}/${apiPath}`;

// Call API after page load
document.addEventListener('DOMContentLoaded', () => {
    loadNews();
});

// Call API to get news data
function loadNews() {
    fetch(apiUrl)
        .then(res => res.json())
        .then(data => {
            renderNews(data);
        })
        .catch(err => {
            console.error('API call failed', err);
        });
}

// Render data to the page
function renderNews(data) {
    const container = document.getElementById('news-container');
    if (!container || !data) return;

    container.innerHTML = data.map(item => `
        <div class="news-item">
            <h3>${item.title}</h3>
            <p>${item.summary || ''}</p>
        </div>
    `).join('');
}
```

index.html

```
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>API Call Example</title>
</head>
<body>

<h1>News List</h1>
<div id="news-container">Loading...</div>

<script src="./main.js"></script>
</body>
</html>

```
