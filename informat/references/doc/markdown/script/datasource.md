# informat.datasource Data Source Related Operations

## Overview

Use `informat.datasource` to perform data source related operations

## getConnection

Get a connection from the database connection pool

```javascript
informat.datasource.getConnection(moduleId, dataSourceKey, setting)
```

::: tip

- An exception will be thrown if the connection cannot be created
- When `autoCommit` is set to `true`, transactions will be automatically committed when executing update, insert, and delete statements
- After script execution, you need to manually close the created connection

:::

| Parameter     | Type                                                                                | Description                    |
|---------------|-------------------------------------------------------------------------------------|--------------------------------|
| moduleId      | `String`                                                                            | Data source module identifier  |
| dataSourceKey | `String`                                                                            | Data source identifier         |
| setting       | [DataSourceConnnectionSetting](/guide/script/model.md#datasourceconnnectionsetting) | Data source configuration      |

***Return Value***
Database connection, type is `JDBCConnection`

**Example**

```javascript
const connection = informat.datasource.getConnection('dataSource', 'db_informat', {
    "autoCommit": true
})
connection.select('select * form table', (row) => {
    console.log(row.getString(1))
})

// Close the connection
connection.close();
```

## Connection

### select

Execute a query. An exception will be thrown if the query fails

```javascript
connection.select(sql, handler, ...args)
```

| Parameter            | Type        | Description              |
|----------------------|-------------|--------------------------|
| sql                  | `String`    | Query SQL                |
| handler(`ResultSet`) | Function    | Query result callback    |
| args                 | `...Object` | Query parameters         |

**Example 1**

```javascript
// Query data from the user table where name equals tom
connection.select('select * from user where name=?', (rs) => {
    console.log(rs.getString('name'))
}, 'tom')
```

**Example 2: Using like**

```javascript
// Query data from the user table where name contains tom
connection.select('select * from user where name like concat(' % ',?,' % ')', (rs) => {
    console.log(rs.getString('name'))
}, 'tom')
```

**Example 3: Using in**

```javascript
// Query data from the user table where name is tom or jerry
connection.select('select name from user where name in(?,?)', (rs) => {
    console.log(rs.getString('name'))
}, 'tom', 'jerry')
```

### insert

Execute an insert. An exception will be thrown if the insert fails

```javascript
connection.insert(sql, returnAutoGeneratedKeys, ...args)
```

| Parameter               | Type        | Description                          |
|-------------------------|-------------|--------------------------------------|
| sql                     | `String`    | SQL statement                        |
| returnAutoGeneratedKeys | `Boolean`   | Whether to return auto-generated primary key ID |
| args                    | `...Object` | Insert parameters                    |

**Return Value**

Type is `String`

If `returnAutoGeneratedKeys` is set to true and the operation is an insert, returns the auto-incremented ID of the last inserted data

**Example**

```javascript
connection.insert('insert into user(name,age) values (?, ?)', true, 'tom', 10)
```

### update

Execute an update or delete. An exception will be thrown if the update or delete fails

```javascript
connection.update(sql, ...args)
```

| Parameter | Type        | Description       |
|-----------|-------------|-------------------|
| sql       | `String`    | SQL statement     |
| args      | `...Object` | Update parameters |

**Return Value**
Type is `Integer`

Returns the number of records successfully updated or deleted

**Example**

::: code-group

```javascript [Update]
connection.update('update user set age=?', 10)
```

```javascript [Delete]
connection.update('delete from user where age=?', 10)
```

:::

### commit

Commit a transaction. An exception will be thrown if the commit fails

```javascript
connection.commit()
```

### rollback

Rollback a transaction. An exception will be thrown if the rollback fails

```javascript
connection.rollback()
```

### close

Close the database connection

```javascript
connection.close()
```
